# DataAnalysisProject

**EN**
In this project you will find real data from expenses made by the Hall Town of Mazatenango City in Suchitepéquez, Guatemala. All documents inside the folder "Compras directas Muni MAzatenango..." are PDFs that were downloaded from the official website of the City Town Hall, they belong to the months of January to May of 2026. Also, there is an excel file with all data that was extracted from those PDF files and then transformed, cleaned, and loaded for analysis.
Finally, the file .pbix contains a visual analysis of all processed data.

-----
**ES**
Este proyecto contiene información real de los gastos a través de compras directas realizados por la alcaldía de la ciudad de Mazatenango en el departamento de Suchitepéquez de Guatemala. Los documentos en la carpeta compras directas Muni Mazatenango son PDFs publicados en la página oficial de la Municipalidad y comprenden los meses de enero a mayo de 2026. También dentro de la carpeta se encuentra un documento de Excel con los datos extraídos de dichos PDFs, los cuales fueron transformados, limpiados y cargados para su análisis.
Finalmente, el archivo .pbix contiene el análisis visual de los datos trabajados.
